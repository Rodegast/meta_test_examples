# meta_test_examples
Примеры тестов на платформе Meta Test.

## demoqa
Пример тестов для сайта https://demoqa.com Требует установку модуля *faker*. Для этого нужно выполнить из каталога дистрибутива:
```
./python/bin/pip3 install faker
```
